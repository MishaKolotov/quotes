## Quotes App

Minimal Angular + Nx demo for random quotes with offline support.

### Requirements

- Node.js **20.19.0** or higher
- npm

### Install dependencies

```bash
npm install
```

### Run the application (dev)

```bash
# starts dev server on http://localhost:4200
npm start
```

### Build the application

```bash
# development build
npm run build

# production build
npm run build:prod
```

### Run tests

```bash
# unit tests (Jest)
npm test
```

### Lint

```bash
npm run lint
```
